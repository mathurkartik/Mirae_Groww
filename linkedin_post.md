🚀 **Excited to share my latest project: Emerald Ledger!** 🚀

Emerald Ledger is a full-stack Mutual Fund Explorer and AI-powered FAQ Assistant designed specifically for Mirae Asset funds. I built this platform to bring an institutional-grade, premium user experience to everyday investors.

Here are some highlights of what I've implemented:
📈 **Context-Aware Return Calculator**: I revamped the generic SIP calculator! The platform now dynamically projects your SIP and lump-sum returns based on the *actual* historical performance (1Y, 3Y, 5Y rates) of the specific fund you're viewing.
🧠 **Facts-Only AI Assistant**: Integrated a custom Retrieval-Augmented Generation (RAG) chatbot using FastAPI, BM25 lexical search, and ChromaDB. It strictly answers questions based on scraped scheme documents—no hallucinated investment advice!
💅 **Premium Fintech UI**: Built with Next.js 15, focusing on a sleek, responsive design with dynamic `recharts` data visualizations and refined micro-animations.
⚙️ **Automated Data Pipelines**: The entire system gets daily fresh data automatically seeded via a robust GitHub Actions scraping pipeline using Python and BeautifulSoup4.

Building this end-to-end (Frontend, Backend APIs, Vector Databases, and CI/CD) has been an incredible journey.
